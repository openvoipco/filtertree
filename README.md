filtertree
==========

Jquery plugin for creating filters on page

After researching web - I have not found a good "ready to go" widget, so - I've wrote it. I18N supported! Hope it will be helpful for someone!

Features:
- I18N
- Nested filters
- Saving state in cookies
- Query builder
